## 文件结构

```
china-cases/
├── index.html      ← 页面结构（一般不改）
├── styles.css      ← 视觉样式（一般不改）
├── app.js          ← 交互逻辑（一般不改）
├── cases.js        ← ★ 案件数据（主要在这里改）
├── i18n.js         ← ★ 文字翻译词典（要改英文/法院名翻译在这里）
```

打 ★ 的是你最常打交道的两个文件。

## 任务二：添加新案件

在 **cases.js** 里复制任意一个 `{...}` 块，粘贴到列表末尾（在最后的 `]` 之前），改成新案件信息：

```javascript
{
  province: "广东省",          // 省份全名（详见下方对照表）
  city: "深圳",                // 法院/城市名（中文）
  title_zh: "案件中文名",
  title_en: "Case English Title",  // 留 "" 也行，会回退显示中文
  caseNumber: "(2026)粤0XXX民初XXX号",
  year: 2026,
  issue: "AIGC 可版权性",      // ★ 七选一，必须完全一致
  tech: ["AI文生图"],          // 详见 i18n.js 里 TECH_I18N
  note_zh: "中文裁判要旨",
  note_en: "English holding",  // 可空 ""
  status: "decided",           // 或 "pending"
  teamArticle: false,          // true=团队有公众号文章
  url: "https://..."           // 原文链接
}
```

**别忘了在前面的 `}` 后面加个 `,`**

## 任务三：修改英文翻译

打开 `i18n.js`，里面有几个词典：

- `I18N.zh` / `I18N.en` —— 界面文字（标题、按钮、标签）
- `COURT_I18N` —— 法院名翻译
- `PROVINCE_I18N` —— 省份名翻译
- `ISSUE_I18N` —— 争议焦点翻译
- `TECH_I18N` —— AI 技术翻译

改对应行，保存，刷新浏览器即可。

**比如想改 "Song Haiyan Team" 为别的写法**：
```javascript
eyebrow: "Song Haiyan Team · Digital Economy & AI",
```
改成你想要的就行。

## 任务四：添加新的法院/城市

cases.js 加新案件时，如果 `city` 字段用了一个新地名（比如"东莞"），需要在 i18n.js 的 COURT_I18N 里加一条翻译：

```javascript
"东莞": "Dongguan Court",
```

如果不加，英文版会显示中文（不会报错，但不规范）。

## 常见问题

**Q: 改了 cases.js 但页面没变化？**  
A: 浏览器有缓存，按 `Cmd+Shift+R`（Mac）或 `Ctrl+Shift+F5`（Windows）强制刷新。

**Q: 页面打不开 / 空白？**  
A: 大概率是 JavaScript 写错了（漏了逗号、引号没闭合）。打开浏览器 → 右键 → 检查 → Console 看红色报错。或者直接发我让我看。

**Q: 案件分类放错了想改？**  
A: 找到 cases.js 里那条案件的 `issue` 字段改成对的就行。

**Q: 想把某个团队评注下掉？**  
A: cases.js 里那条案件的 `teamArticle` 改成 `false`。
